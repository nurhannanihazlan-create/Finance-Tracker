# Project Title
Personal Finance Tracker

## Project Description
A web-based personal finance tracker that allows users to view income, expenses, commitments and financial summary.

## Features
- Login/Register (simulated authentication)
- Dashboard with summary cards
- Income and Expenses data tables
- Data visualization using Chart.js
- Responsive UI using Bootstrap 5
